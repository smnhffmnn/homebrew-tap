# Homebrew Tap

Homebrew formulae for [mux](https://github.com/smnhffmnn/mux) and [vox](https://github.com/smnhffmnn/vox).

## Installation

```bash
brew tap smnhffmnn/tap
brew install mux
brew install vox
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [mux](https://github.com/smnhffmnn/mux) | Single-binary MCP gateway for databases, APIs, and tunnels |
| [vox](https://github.com/smnhffmnn/vox) | Speech-to-text dictation tool with context-aware cleanup |
